Minesweeper

- Individual Work

I used Java to redesign a classical game "Minesweeper" with additional funcitons:
• Users can choose to listen to different background music and enable or disable it.
• There are sound effects when users win or lose the game.
• Three chances of help to open the block which hides a bomb. Then users can put a flag on it.
• Pause function for user to pause the game if they need to leave for a while.

Maze

- Team work with Doris

Happy Kitty is a customized maze game designed for kids.

• Players can customize the width and height of the maze map.
• Players can input key, helmet and clock wherever they want within the map.
• In order to win, players have to move the kitty to the exit door of maze with a key.
• Players can pick up the key, helmet and clock when they encounter with them.

I designed the interface of the game and developed the main function by Java.

Gamification Question

-Team work with Jérôme Cabridens

Gamification Question is a game to guess users’ emotions by questions guiding.

• At first, users can choose one situation, then corresponding questions will be given to users.
• By answering those questions, users’ emotion will be detected step by step.
• Finally, the system will show a facial emotion to users based on the situation they choose.

• The architecture of this game is based on OCC model revisited. We used processing to implement this game.
